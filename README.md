Machine learning project developed for the AI Club DC Hackathon 2024–25 to predict UEFA Champions League knockout-stage winners using historical match data.

# What I Did
1.Processed historical UEFA Champions League data
2.Engineered team-performance and tournament-specific features
3.Trained and compared multiple ML models
4.Built an ensemble using weighted soft voting
5.Generated predictions for future knockout-stage matchups

#Models Used
Logistic Regression
Random Forest
Gradient Boosting

#Results
The ensemble approach provided more robust predictions than individual models and achieved the best ranking performance on the validation set.
